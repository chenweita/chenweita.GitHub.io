1. JSONP封装
JSONP封装是指将JSONP技术进行代码抽象和模块化处理，以简化跨域数据请求的调用流程。JSONP通过动态创建script标签实现跨域，但原生方式需手动处理回调函数和错误，封装后可提升开发效率和代码可维护性。‌

2. 实现思路
    1. 函数接受两个参数，一个请求的url，一个回调函数
    2. 创建一个script标签，设置window.cbname = {
        执行回调函数，清除此属性，清除script标签
    }
    3. 设置script标签的src属性并且插入

3. 代码示例
    function jsonP(url, cb) {
        let cbName = `jsonCB_${Date.now()}`;
        let script = document.createElement('script')
        window[cbName] = (data) => {
            cb(data);
            document.body.removeChild(script)
            delete window[cbName]
        }
        let rqUrl = `${url}?cb=${cbName}`
        script.src = rqUrl;
        document.body.appendChild(script);
    }
